# **App Name**: BotSpeak Insights

## Core Features:

- Dynamic Recording & Question Presentation: Fetches and displays audio recordings along with their associated questions dynamically from a backend database, enforcing a sequential, unskippable progression through assessments.
- Audio Playback Interface: Provides a user-friendly interface with controls (play, pause, seek) for listening to each bot conversation recording.
- Star Rating Input: Enables users to provide a 1-5 star rating for how human-like they perceive the bot's sound.
- Open-ended Text Response: Offers a multi-line text input area for users to articulate detailed feedback on their assessment of the bot's human-likeness.
- Assessment Progression Management: Controls the workflow to ensure that users complete the questions for one recording before being allowed to proceed to the next.
- Final User Contact Form: Presents a form to collect the user's name and email address after all recordings and questions have been successfully completed.
- Assessment and Contact Data Storage: Securely stores all user assessment responses and final contact information within the Firebase database.

## Style Guidelines:

- Primary color: A deep, professional blue (#1F66AD) to convey trust and clarity.
- Background color: A very light, neutral blue-gray (#F0F2F4) to create a clean and focus-oriented canvas.
- Accent color: A vibrant, refreshing cyan (#30E8E8) used for interactive elements and highlights to draw attention.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, scientific feel, while body text will use 'Inter' (sans-serif) for objective readability.
- Use minimalist and intuitive icons for audio playback controls, star ratings, and progression indicators to maintain a professional aesthetic.
- A clean two-column layout as depicted in the diagram, with the left column displaying recording progression and the right column dedicated to the active assessment form, ensuring clear separation and flow.
- Subtle and smooth transitions for moving between recordings and sections of the assessment form to enhance user experience without distraction.
